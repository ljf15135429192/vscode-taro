# cssrem

A `px` to `tow px` and taro snippet plugin in VSCode



# Install

```bash
ext install taro
```

# How To Use

+ Auto snippet
+ CLI: Press `F1`, enter `taro`

# Support Language

html vue css less scss sass stylus tpl（php smarty3）

# Configuration

暂无

**NOTE:** Muse be restart vscode after modification
